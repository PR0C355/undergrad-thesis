A Large-Scale Training Dataset for Temporal Grounding in Long-Form Video

ChronoQA

Timestamp-Anchored Multiple-Choice QA Pairs for Training Temporally Grounded Video-Language Models
Oluwatumininu Oguntola  ·  Pranav Wagh  ·  Gedas Bertasius
University of North Carolina at Chapel Hill  |  2026
Motivation

Current video-language models (VLMs) are trained on data that provides answer supervision but lacks temporal evidence supervision. Without knowing where in a video an answer is grounded, models learn shortcuts instead of genuine temporal reasoning.

Gemini 3 Pro scores 88.4% on Video-MME yet only 48.2% on fine-grained temporal tasks — a gap traced directly to the absence of temporal grounding signal in training data.

ChronoQA addresses this at the source: every training example pairs a question and correct answer with a verified timestamp interval, giving models an explicit supervision target for when the answer occurs — not just what the answer is.

The Temporal Grounding Gap

Training without timestamp supervision produces models that exploit:

1
Static shortcuts — single-frame cues that happen to correlate with correct answers
2
Linguistic priors — matching question text to answer text without video engagement
3
Positional bias — dataset artifacts that cluster evidence near video boundaries

ChronoQA's training examples are explicitly designed to close each of these loopholes.

Dataset Structure

Each ChronoQA instance provides three levels of supervision simultaneously: the correct answer, the supporting timestamp interval, and four semantically plausible distractors — enabling both answer and grounding losses during training.

40K+source videos
5–60min per video
5-wayMCQ + timestamps
Dual supervision per example: answer label (A–E) and a verified [t_start, t_end] interval — enabling models to be trained jointly on answer correctness and temporal localization quality.
Five Training Categories

Ordered by temporal scope — from segment-level to full-video reasoning

Action Understanding
Fine-grained segment content — tool usage, body posture, procedural steps. Single-segment scope.
Needle-in-Haystack
Specific visual details (brands, colors, on-screen text) distributed uniformly across the full timeline to prevent positional bias.
Temporal Ordering
Event sequencing across minutes-long gaps — before/after relationships, transitions between named states.
Causal Understanding
Cross-segment synthesis — no single segment suffices; models must integrate evidence from multiple disjoint parts.
General Understanding
Full-video holistic comprehension — overarching narrative, primary subject, tone. Correct answers require processing the entire video; no single segment is sufficient.
Training-Oriented Design Principles
A
No temporal leakage in questions — questions never reference "when," "beginning," or timestamps, so models cannot learn surface-form shortcuts from question text.
B
Uniform answer position distribution — correct answers are balanced across A–E, preventing models from learning positional biases.
C
Semantically grounded distractors — wrong options are plausible events at incorrect timestamps (or correct timestamps with wrong descriptions), forcing contrastive temporal reasoning.
D
Blind-filtered for visual necessity — text-only models cannot exceed random chance (~20%), ensuring every example genuinely requires visual temporal engagement.
Scalable Generation Pipeline
YouTube Crawl
5–60 min videos; Creative Commons licensing; CLIP-based visual diversity filter
↓
Segment & Caption
10s clips via Qwen3-VL-30B at 1 fps; timestamped free-form captions per segment
↓
QA Generation
Qwen3-Next-80B; structured JSON outputs; 3 Qs per category per video; ~98% success rate
↓
Validation & Cleaning
Temporal pattern removal; timestamp merging; jitter ±1–3s for annotation variance
↓
Distractor Generation
4 semantically plausible distractors grounded in ±30s local context window
↓
Blind Filtering
Multiple text-only LLMs discard linguistically solvable examples; retains visual-only Qs

Fully automated — scalable to hundreds of thousands of examples with no manual annotation.

Training Signal per Instance
Answer supervision Label ∈ {A, B, C, D, E} → cross-entropy loss
Temporal grounding supervision Target: [t_start, t_end] per correct answer
tIoU = |P ∩ G| / |P ∪ G| → grounding loss
Joint training objective L = L_answer + λ · L_grounding

The dual-objective formulation trains models to be correct and to localize evidence — directly targeting the failure mode where high QA accuracy masks poor temporal grounding.

Downstream Use Cases
Supervised fine-tuning
Train VLMs on answer + timestamp pairs to instill temporal grounding as a learned capability.
RLHF / preference data
Use grounded vs. ungrounded correct answers as preference pairs for reward modeling.
Pretraining augmentation
Mix into video pretraining corpora to improve temporal awareness from the ground up.
Probing & diagnostics
Held-out split supports evaluation of temporal grounding across five reasoning categories.
